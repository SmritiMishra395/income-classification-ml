# Income Classification using Machine Learning

## Project Overview

This project predicts whether a person's income is **>50K or <=50K** using the Adult Census dataset.

## Dataset

Adult Census Income Dataset from UCI / Kaggle.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn

## Model Used

Logistic Regression

## Steps in the Project

1. Data preprocessing
2. Handling missing values
3. Encoding categorical variables
4. Train-test split
5. Model training
6. Model evaluation

## Output

The model predicts the income category and evaluates performance using accuracy and confusion matrix.
